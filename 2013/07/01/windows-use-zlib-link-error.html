Windows下编译使用zlib库的程序时出现“无法解析的外部符号 _compress”解决方案

2013-07-01

文章目录
    • 错误描述
    • 解决方案

错误描述

error LNK2019: unresolved external symbol _compress referenced

解决方案

在 #include <zlib.h> 之前加上 #define ZLIB_WINAPI

#define ZLIB_WINAPI
#include <zlib.h>
