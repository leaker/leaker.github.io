error LNK2026: module unsafe for SAFESEH image 解决方案

2013-10-03

文章目录
    • 解决方案
      • 方法1
      • 方法2

解决方案

下面两种随便选一种

方法1

去掉项目设置中: Linker -> Advanced -> [Image Has Safe Exception Handlers] = “No”

方法2

直接在代码中:

#pragma comment(linker, "/SAFESEH:NO")
