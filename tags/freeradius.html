解决编译freeradius出现 cannot find -lperl 的问题

2012-03-27

今天因为项目需要，需要编译freeradius来使用。

错误原因

结果在编译过程中出现如上错误 cannot find -lperl。

解决方案

CentOS

sudo yum -y install libtool-perl*

Ubuntu

sudo apt-get install libperl-dev
